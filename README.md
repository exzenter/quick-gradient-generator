# Quick Gradient Generator

A simple web-based tool for generating beautiful CSS gradients quickly and easily.

## Live Demo

Visit the live application at: https://exzenter.github.io/quick-gradient-generator/

## Features

- Generate random gradient combinations
- Customize colors and gradient directions
- Copy generated CSS code
- Real-time preview

## Usage

Simply open the application in your web browser and start creating gradients. No installation required.
